# CI Reader

This repo now hosts a static sentence-by-sentence Chinese reader for **The Legendary Moonlight Sculptor**.

## Included in this repo

- A chapter picker for the first 3 pilot chapters
- One Chinese sentence at a time
- Clickable highlighted words with pinyin and English gloss
- A button to reveal the full sentence in helper English
- Local progress saving in the browser
- GitHub Pages deployment workflow

## Site layout

- `site/index.html`
- `site/styles.css`
- `site/app.js`
- `site/data/content.json`
- `.github/workflows/deploy-pages.yml`

## Notes

This is a static deployment-oriented version of the reader so the project is visible and usable on GitHub right away.
